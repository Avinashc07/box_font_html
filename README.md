# Responsive Image Display Page

## Overview
This project is a simple responsive web page built using HTML and CSS. It displays an image inside a container with a styled gradient button. The layout is designed using viewport units to ensure responsiveness and prevent page scrolling.

## Features
- Responsive HTML5 structure
- External CSS stylesheet
- Image displayed inside a container
- Gradient-styled button
- Flexbox layout
- Uses viewport units (vw, vh)
- Proper use of `box-sizing`
- No page scrolling (`overflow: hidden`)
- Clean and simple design

## Technologies Used
- HTML5
- CSS3
